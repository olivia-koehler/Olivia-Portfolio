Projects

Sleepsta is a sleep tracking app that allows users to track their sleep motion using their mobile phones, and view that data on the web app. This was a five-week project where I helped create the front-end and back-end with a team using agile methodologies. On the back-end I helped plan our data tables, and created multiple endpoints for receiving, adding, and updating data. On the front-end I mainly dealt with the graphs on the dashboard, letting users view and sort their sleep data by date.

Tech: React, Redux, Node.js, Express, Knex, canvasJS, apexcharts, Reactstrap, Styled-Components

Website GitHub

Use-My-Tools is a website that allows users to login and select tools to borrow from other users, or create listings for tools for others to borrow. I created the entire back-end for this project during a project week at Lambda School, where I worked with a team using agile methodologies.

Tech: Node.js, Express, PostgreSQL, Knex, Jest

Website GitHub

TipsEase is a website that allows customers to tip employees directly. This was created during a project week at Lambda School, where I worked with a team using agile methodologies. I was responsible for creating the front-end of the website.

Tech: React, Redux, Styled-Components

Website GitHub

Insta-Clone is based on the Instagram website. The app includes the ability to login, search posts, and add comments.

Tech: React, Styled-Components, Higher-Order Components.

Website GitHub

S&J Architects is a fully-responsive, multi-page marketing website I created during a project week at Lambda School.

Tech: HTML5, CSS3 (LESS), Javascript, GSAP
